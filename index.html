MTG Decks & Inventory

A plain-text friendly homepage for browsing decks, spare-card inventory, and project config on GitHub Pages.

Deck Browser Inventory README Functional Spec Valuation Report

Decks

Open the Deck Browser to read decklists in their original markdown. Each deck is loaded from the repository so you can keep editing the files you already maintain.

Inventory

The Inventory page parses the inventory.md table and lets you search, sort, and filter cards while keeping the black background and white text aesthetic.

Configuration & Notes

Quick links to documentation and reports live alongside the decks and inventory so everything is reachable from one index page.
